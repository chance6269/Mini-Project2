1. 프로젝트 개요 및 목적

 - 목적 : 부동산 가격 예측 알고리즘을 개발하여 고객의 부동산 거래에 도움을 제공하는 것을 목표로 함.
 - 개요 : 
	A. 웹사이트 혹은 어플리케이션은 고객이 지역을 선택하면, 해당 지역의 실거래지수 등락을 예측하여 제공
		B1. 부동산 매물 API -> 매물정보 읽고 매물의 미래가격 예측
		https://brunch.co.kr/@clay1987/315
	
		B2. 도시 양극화 분석
			
			- 실거래 지수 활용
			- 세종, 대전 vs 충남, 충북
				목표 : 세종, 대전에 비해 타도시들이 부족한 것들을 해소하여 균형 발전을 도모
				
			- 양극화 지표 변수들 :
				- 매매실거래지수
				
			- 원인이 되는 변수들 :
				- 주택 보급률
				- 노후주택비율
				- 의료 서비스 인구 밀도
				- 녹지율
				- 생산가능인구비율

	플랜 A:
	
		# ROW DATA :
		 
		 - 종속 변수(Y) : 지역별 아파트 매매 실거래지수 데이터
		 - 독립 변수(X) :
		 
			- 지역별 인구 이동 데이터				
			- 기준금리 vs 주택담보대출금리:
				- 0.86으로 서로 강한 양의 상관관계를 보임으로, 다중공선성 문제 해소를 위해
					실거래지수와 상관관계가 높은 주택담보대출금리 채택
			- PIR지수 vs LIR지수:
				- 둘이 서로 다른 측면에서 주택 시장의 *가용성(affordability)을 평가하나, 0.96으로 상관관계가 매우 높아
					실거래지수와 상관관계가 높은 PIR지수 채택
					* 일반적으로 개인이나 가구가 주택을 구매하거나 임대하는 데 필요한 비용을 수용할 수 있는 정도
					
			- 소비자물가지수 :
				- 상관계수 0.82로 높은 편
				
			- 아파트매매거래량
			
			- 종합부동산세 세율(중위값) :
				- 상관계수 0.897로 높은 상관계수 확인
				
			- 매매수급동향 : 
				- 한국부동산원(구.한국감정원)에서 제공하는 전국주택가격동향 월간동향 중 아파트의 수급동향-매매수급동향 데이터
				- 0 (공급우위) ~ 200 (수요우위)
						
						
			* 탈락한 변수들 :
				- 지역별 1인 가구 수
					- 높은 상관관계를 확인했으나 월별데이터가 없어서 빠른 반영 불가 -> 변수로 사용 불가
					
				- 국회의원 의석수 우위(보수 vs 진보)
					- One-Hot Encoding하여 넣어봤으나, 예측율이 감소함.
					
				- 주택구입부담지수
					1) 월별 데이터가 아닌 분기 데이터만 확보가능  -> 빠른 반영 불가 -> 변수로 사용 불가
